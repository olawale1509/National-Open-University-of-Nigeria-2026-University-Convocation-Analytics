# NOUN Convocation Analysis Dashboard
National-Open-University-of-Nigeria-2026-University-Convocation-Analytics

# Project Overview
This project analyses convocation data for the National Open University of Nigeria using Power BI across faculties, programmes, and study centres. It tracks student performance metrics, highlights key insights like a critical 0.23% first-class rate, and delivers actionable recommendations to improve academic support, faculty balancing, and institutional growth

# Business/Academic Problem
Large higher-education institutions often struggle to visualise student outcomes across widespread geographical study centres and diverse academic programmes. This project transforms raw graduation data into actionable insights, specifically addressing structural challenges such as low top-tier graduation rates and faculty enrollment imbalances.

#🛠️ Tools & Techniques Used
Business Intelligence: Microsoft Power BI extracted from Microsoft Excel
Data Processing: Power Query for data cleaning, transformation, and structural formatting.
Data Modelling: Built relational models connecting Convocation Tables, Study Centres, and Approval Dates.
Calculations: Developed DAX (Data Analysis Expressions) measures for custom KPIs, percentage rate calculations, and dynamic filtering.
Data Visualisation: Utilised advanced mapping, matrix grids, dynamic KPI cards, and custom tooltips to ensure an intuitive user experience.

## Analysis Performed
- Programme analysis
- Faculty analysis
- Study centre analysis
- State distribution analysis
- Yearly convocation trend analysis

#📊 Key Insights & Recommendations
Through interactive data visualisation, this project uncovered several critical academic trends:
Academic Performance Gaps: Identified a critically low first-class graduation rate (0.23%), prompting a recommendation for targeted mentorship and academic support programs.
Programme Concentration: Revealed that the massive majority of graduates (71.24%) concentrated in Bachelor's degrees, indicating a strategic need to boost Postgraduate enrollment for institutional growth.
Geographical Distribution: Mapped graduation densities across national study centres, like Lagos, Abuja, and Port Harcourt, highlighting regions with the highest output and standardising academic performance tracking across locations.
