Project Overview
This Power BI project provides a comprehensive analysis of university convocation data, tracking over 25,000 graduating students across various faculties, degree programmes, and regional study centres. The dashboard is designed to act as an institutional research tool, helping academic administrators and policymakers identify performance gaps, optimize programme offerings, and balance faculty enrollment.

Business / Academic Problem
Large higher-education institutions often struggle to visualize student outcomes across widespread geographical study centres and diverse academic programmes. This project transforms raw graduation data into actionable insights, specifically addressing structural challenges such as low top-tier graduation rates and faculty enrollment imbalances.

📊 Key Insights & Recommendations
Through interactive data visualization, this project uncovered several critical academic trends:
Academic Performance Gaps: Identified a critically low first-class graduation rate (0.23%), prompting a recommendation for targeted mentorship and academic support programs.
Programme Concentration: Revealed that a massive majority of graduates (71.24%) are concentrated in Bachelor's degrees, indicating a strategic need to boost Postgraduate enrollment for institutional growth.
Geographical Distribution: Mapped graduation densities across national study centre like Lagos, Abuja, Port Harcourt, highlighting regions with the highest output and standardizing academic performance tracking across locations.

🛠️ Tools & Techniques Used
Business Intelligence: Microsoft Power BI extracted form Microsoft Excel
Data Processing:Power Query for data cleaning, transformation, and structural formatting.
Data Modeling: Built relational models connecting Convocation Tables, Study Centres, and Approval Dates.
Calculations: Developed DAX (Data Analysis Expressions) measures for custom KPIs, percentage rate calculations, and dynamic filtering.
Data Visualization: Utilized advanced mapping, matrix grids, dynamic KPI cards, and custom tooltips to ensure an intuitive user experience.
